# 心情论坛 (Heart Forum)

一个温暖、治愈的在线交流社区，帮助人们分享心情、倾诉烦恼、互相支持。

## 功能特性

### 用户功能
- 用户注册/登录/退出
- 帖子发布与管理
- 评论回复
- 点赞功能（帖子和回复）
- 私信系统
- 个人资料管理
- 消息通知

### 管理员功能
- 用户管理（封禁/解封）
- 帖子管理（置顶/锁定/删除）
- 分类管理
- **违禁词管理**（批量导入、搜索过滤、内容测试）
- **举报处理**（统计、批量操作、处理备注）
- **内容审核**（审核队列、通过/拒绝、审核记录）
- 数据统计看板

## 技术栈

- **后端**: Flask
- **数据库**: SQLite
- **认证**: Flask-Login
- **表单**: Flask-WTF
- **前端**: Bootstrap 5 + Jinja2

## 快速开始

### 环境要求
- Python 3.8+

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行应用

```bash
python app.py
```

应用将在 http://127.0.0.1:5000 启动

### 默认管理员账户

- 用户名: `admin`
- 密码: `admin123`

首次运行会自动创建数据库和默认数据。

## 项目结构

```
forum/
├── app.py              # 主应用文件
├── models.py           # 数据模型
├── forms.py            # 表单定义
├── templates/          # Jinja2 模板
│   ├── base.html       # 基础模板
│   ├── auth/           # 认证相关
│   ├── forum/          # 论坛功能
```

## 核心功能说明

### 违禁词管理
- 支持添加单个违禁词
- 支持批量导入（换行或逗号分隔）
- 实时内容测试，检查文本是否包含违禁词
- 关键词过滤功能

### 举报处理
- 用户可举报帖子和回复
- 管理员查看并处理举报
- 支持删除内容或封禁用户
- 批量处理功能

### 内容审核
- 审核队列管理
- 审核通过/拒绝操作
- 记录审核人员和审核时间
- 可选择是否删除内容
